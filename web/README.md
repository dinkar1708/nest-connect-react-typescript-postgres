# NestConnect Web

**Project:** nestjs-react-typescript-friend-service

React + TypeScript + Vite frontend for NestConnect.

**Technology stack**

| Layer | Tech |
|-------|------|
| Framework | React v19 |
| Language | TypeScript |
| Build Tool | Vite v7 |
| Styling | CSS |
| Linting | ESLint |

**SEO** — No SSR (Client-side only) - Not suitable for content-heavy/SEO-critical apps. To achieve SEO: Migrate to Next.js (SSR/SSG), use Remix, or implement react-snap for pre-rendering.

**Start**
```bash
npm install
npm run dev
```

**Test the UI**

Open [http://localhost:5173](http://localhost:5173) in your browser after running `npm run dev`.

**Build**
```bash
npm run build
```
