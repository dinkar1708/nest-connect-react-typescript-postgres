# NestConnect Web

**Project:** nestjs-react-typescript-friend-service

React + TypeScript + Vite frontend for NestConnect.

**Technology stack**

| Layer | Tech |
|-------|------|
| Framework | React 19 |
| Language | TypeScript |
| Build | Vite |
| Styling | CSS |
| Lint | ESLint |

**SEO** — Client-side SPA; crawlers must execute JS. To improve: add meta description and Open Graph in `index.html`; use `react-helmet-async` for per-page meta; or migrate to SSR (Next.js/Remix).

**Start**
```bash
npm install
npm run dev
```

**Test the UI**

Open [http://localhost:5173](http://localhost:5173) in your browser after running `npm run dev`.

**Build**
```bash
npm run build
```
